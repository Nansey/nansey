SDL 机器翻译新增新的智能功能

2021-02-26

RWS 集团旗下的 SDL 今天宣布，其机器翻译技术增加了新的智能功能和能力，包括内容洞察（content insight）、采用扩展选项（expanded options for adaptation）和自动质量评估（automated quality assessment），为翻译公司提供了更智能的方式来服务全球客户。

SDL 于2020年11月被 RWS 收购。RWS 为全球百强品牌中的90家、全球前10制药公司和全球前20家专利公司中的半数提供服务。
